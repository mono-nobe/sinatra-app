# TODO APP

## About

TODOアプリです。やることリストを作成・編集・削除することができます。

## Getting Started

1. リポジトリをクローンします。

```sh
$ git clone https://github.com/mono-nobe/sinatra-app.git sinatra-app
```

1. 作業ディレクトリへ移動します。

```sh
$ cd sinatra-app
```

1. Gemをインストールします。Bundlerのバージョンは2.2.32です。

```sh
$ bundle install
```

1. アプリを起動します。

```sh
$ ruby app.rb
```

1. ブラウザで以下のURLへアクセスします。

```txt
http://localhost:4567/todos
```
